## Getting Started

### Quick Start Guide

1. **Visit the Website** — Go to [https://zeroupload.io](https://zeroupload.io)
2. **Choose a Tool** — Browse categories or search for specific functionality
3. **Upload File(s)** — Select file(s) from your computer
4. **Configure Options** — Adjust settings as needed
5. **Process** — Click the process button
6. **Download** — Save your processed file

### Key Principles

- **100% Private** — All data stays on your computer
- **No Registration** — Use instantly without account creation
- **No Server Uploads** — Files never leave your browser
- **Fast Processing** — Operations happen locally
- **Always Free** — No hidden costs or premium features

### Browser Requirements

- **Minimum:** Any modern browser (Chrome, Firefox, Safari, Edge)
- **Recommended:** Latest version of your preferred browser
- **JavaScript:** Must be enabled
- **Storage:** For large files, sufficient RAM available

### 3. Hash Generator

**Purpose:** Create hash values from text

#### Supported Hashing Algorithms
- **MD5** — 128-bit hash (legacy, not secure)
- **SHA-1** — 160-bit hash (deprecated for crypto)
- **SHA-256** — 256-bit hash (recommended)
- **SHA-512** — 512-bit hash (high security)

#### How to Use
1. Paste text to hash
2. Select algorithm
3. View output hash
4. Copy hash
5. Use in your application

#### Hash Examples

```
Text: "Password123"

MD5:     482C811DA5D5B4BC6D497FFA98491E38
SHA-1:   3EBBED8B8D1B4B41FA27FA58BCDE0E11AB8C1F68
SHA-256: 0B1DF65F1B1B1B1B1B1B1B1B1B1B1B1B1B1B1B1B
SHA-512: 1234567890ABCDEF... (longer)
```

#### When to Use

| Algorithm | Use | Status |
|-----------|-----|--------|
| MD5 | Checksums only | Legacy |
| SHA-1 | Git, legacy systems | Deprecated |
| SHA-256 | Passwords, security | Recommended |
| SHA-512 | High security | Recommended |

#### Security Notes
- Hashing is **one-way** (can't reverse)
- Use SHA-256 or higher for passwords
- Multiple same inputs = same hash (use salt for passwords)
- Never rely on MD5 for security

#### Tips
- Use SHA-256 as default choice
- Save hash for file verification
- Combine algorithms for extra security
- Never store plaintext with hashes

## Troubleshooting

### Common Issues & Solutions

#### Issue: File Won't Upload

**Solutions:**
1. Check file size (large files may timeout)
2. Verify file format is supported
3. Try refreshing page
4. Clear browser cache
5. Use different browser

**Technical Details:**
- Browser memory limit: typically 1-2GB
- Large files require more processing time
- Some formats have specific size limits

#### Issue: Processed File Corrupted

**Solutions:**
1. Verify source file wasn't corrupted
2. Try intermediate compression
3. Use different tool/method
4. Download again from browser
5. Consult tool-specific documentation

**Prevention:**
- Keep backup of original file
- Process one format at a time
- Verify downloaded file immediately
- Test with small file first

#### Issue: Theme Not Saving

**Solutions:**
1. Check localStorage is enabled
2. Verify browser cookies/storage permitted
3. Check browser privacy settings
4. Try different browser
5. Clear cache and try again

**Browser Settings:**
- Settings → Privacy → Allow storage
- Ensure cookies are not blocked
- Check third-party script blocking

#### Issue: Tool Not Responding

**Solutions:**
1. Check internet connection
2. Wait for processing to complete
3. Try refreshing page
4. Clear browser cache
5. Restart browser

**For Large Files:**
- Processing may take several minutes
- Do not close tab while processing
- Monitor browser console for errors
- Try smaller file first

#### Issue: Download Interrupted

**Solutions:**
1. Try downloading again
2. Use different browser
3. Check available disk space
4. Disable download protection
5. Verify internet connection

**Prevention:**
- Ensure stable connection
- Check disk space before processing
- Don't close browser during download
- Avoid concurrent large downloads
